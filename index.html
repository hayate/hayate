Current Features:
  • fast, small, essential
  • out of the box smarty and native templates support
  • HTTP put method support, (useful for REST api)
  • Validator class
  • modules support
  • strictly OOP
  • classes autoloading
In Development
  • Database support via PDO
  • solid ORM

Why another PHP framework ?

Because is fun, is much more fun and rewarding building sites/CMS with your own framework then with any of the frameworks currently out there.
